CSS Syntax Review

Remember: CSS follows this structure:

selector {
    property: value;
}

Three Ways to Add CSS

  1. Inline CSS: Using the style attribute directly in HTML
  2. Internal CSS: Using a <style> tag in the <head> section (like this file)
  3. External CSS: Linking to a separate .css file
